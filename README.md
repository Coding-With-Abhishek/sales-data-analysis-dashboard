# Sales Data Analysis Dashboard

## Overview

This project analyzes sales data and provides insights into:

- Total Sales
- Total Profit
- Top Selling Products
- Regional Performance
- Category Performance

## Technologies Used

- Python
- Pandas
- Matplotlib
- CSV Dataset
- GitHub

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Data Analysis
4. Data Visualization
5. Business Insights

## Features

### KPI Analysis

- Total Sales
- Total Profit

### Business Insights

- Best Performing Region
- Top Products
- Sales Trend

### Visualization

- Category-wise Sales Chart
